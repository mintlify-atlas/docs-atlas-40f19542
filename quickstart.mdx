title: "Quickstart — First speech recognition in 5 minutes"
description: "Learn how to build a working speech recognition component in React Native with expo-speech-recognition using hooks and the module API."
---

In this quickstart you will build a minimal but fully working React Native component that requests microphone and speech recognition permissions, listens for spoken words, and displays the live transcript on screen. By the end you will have a solid foundation to explore more advanced features like continuous recognition, audio recording, and on-device transcription.

<Steps>
  <Step title="Install and configure the library">
    Follow the [installation guide](/installation) to add `expo-speech-recognition` to your project, configure the Expo config plugin in `app.json`, and rebuild the native projects:

    ```bash
    npx expo run:android
    # or
    npx expo run:ios
    ```

    <Note>
      expo-speech-recognition requires a development build — it will not run inside Expo Go.
    </Note>
  </Step>

  <Step title="Request permissions">
    Before calling `start()`, ask the user for microphone and speech recognition permissions. Call `requestPermissionsAsync()` — it covers both the microphone (`RECORD_AUDIO` on Android) and the speech recognizer (`NSSpeechRecognitionUsageDescription` on iOS) in a single call.

    ```ts
    import { ExpoSpeechRecognitionModule } from "expo-speech-recognition";

    const result = await ExpoSpeechRecognitionModule.requestPermissionsAsync();

    if (!result.granted) {
      console.warn("Permissions not granted", result);
      return;
    }

    // Safe to call start() now
    ```

    If the user denies permissions, any subsequent call to `start()` will fire an `error` event with the code `"not-allowed"`. You can check the current permission status at any time with `getPermissionsAsync()` without prompting the user again.
  </Step>

  <Step title="Subscribe to recognition events">
    Use the `useSpeechRecognitionEvent` hook to register listeners for the events you care about. The hook automatically cleans up the listener when the component unmounts.

    ```ts
    import { useSpeechRecognitionEvent } from "expo-speech-recognition";

    // Called when the recognizer has started and is ready for input
    useSpeechRecognitionEvent("start", () => setRecognizing(true));

    // Called when the session ends (always fires last, even after errors)
    useSpeechRecognitionEvent("end", () => setRecognizing(false));

    // Called whenever a partial or final result is available
    useSpeechRecognitionEvent("result", (event) => {
      setTranscript(event.results[0]?.transcript);
    });

    // Called when an error occurs
    useSpeechRecognitionEvent("error", (event) => {
      console.log("error code:", event.error, "error message:", event.message);
    });
    ```

    The event names and `error` codes mirror the [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/SpeechRecognition) so the mental model is familiar if you have used browser-based speech recognition before.
  </Step>

  <Step title="Start and stop recognition">
    Call `ExpoSpeechRecognitionModule.start()` with a `lang` option (BCP-47 locale tag) and any additional options you need. Call `stop()` to request a graceful end — the recognizer will attempt to return a final result before shutting down. Call `abort()` to cancel immediately without waiting for a result.

    ```ts
    import { ExpoSpeechRecognitionModule } from "expo-speech-recognition";

    // Start
    ExpoSpeechRecognitionModule.start({
      lang: "en-US",
      interimResults: true,  // receive partial results as the user speaks
      continuous: false,     // stop after the first final result
    });

    // Stop gracefully (emits a final result if possible)
    ExpoSpeechRecognitionModule.stop();

    // Cancel immediately (no final result)
    ExpoSpeechRecognitionModule.abort();
    ```
  </Step>
</Steps>

## Complete example

The component below combines all four steps into a working app. Copy it into a new file in your project and render it from your root component.

```tsx
import { useState } from "react";
import { Button, ScrollView, Text, View } from "react-native";
import {
  ExpoSpeechRecognitionModule,
  useSpeechRecognitionEvent,
} from "expo-speech-recognition";

function App() {
  const [recognizing, setRecognizing] = useState(false);
  const [transcript, setTranscript] = useState("");

  useSpeechRecognitionEvent("start", () => setRecognizing(true));
  useSpeechRecognitionEvent("end", () => setRecognizing(false));
  useSpeechRecognitionEvent("result", (event) => {
    setTranscript(event.results[0]?.transcript);
  });
  useSpeechRecognitionEvent("error", (event) => {
    console.log("error code:", event.error, "error message:", event.message);
  });

  const handleStart = async () => {
    const result = await ExpoSpeechRecognitionModule.requestPermissionsAsync();
    if (!result.granted) {
      console.warn("Permissions not granted", result);
      return;
    }
    ExpoSpeechRecognitionModule.start({
      lang: "en-US",
      interimResults: true,
      continuous: false,
    });
  };

  return (
    <View>
      {!recognizing ? (
        <Button title="Start" onPress={handleStart} />
      ) : (
        <Button
          title="Stop"
          onPress={() => ExpoSpeechRecognitionModule.stop()}
        />
      )}
      <ScrollView>
        <Text>{transcript}</Text>
      </ScrollView>
    </View>
  );
}
```
